# 📋 Job Tracker Frontend

A modern job application tracking dashboard built with Next.js 14, TypeScript, and Tailwind CSS.


---

## ✨ Features

- 📊 **Dashboard Statistics** - View total applications and counts by status
- 🔍 **Search** - Search by company name or job title
- 🏷️ **Filter** - Filter by status (Applied, Interviewing, Offer, Rejected)
- ➕ **Create** - Add new job applications
- ✏️ **Edit** - Update existing applications
- 👁️ **View** - Detailed view of each application
- 🗑️ **Delete** - Remove applications with confirmation

---

## 🛠 Tech Stack

| Category | Technology |
|----------|------------|
| **Framework** | Next.js 14 (App Router) |
| **Language** | TypeScript |
| **Styling** | Tailwind CSS |
| **API Client** | Axios |
| **Forms** | React Hook Form + Zod |
| **Icons** | React Icons |
| **Date Handling** | date-fns |
